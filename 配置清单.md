# 启动前需要准备的东西

## 1. LLM API Key（二选一）

### 选项 A：Anthropic Claude（推荐）
- 获取地址：https://console.anthropic.com/settings/keys
- 填入 `.env` 的 `ANTHROPIC_API_KEY`
- 确保 `LLM_PROVIDER=claude`

### 选项 B：OpenAI
- 获取地址：https://platform.openai.com/api-keys
- 填入 `.env` 的 `OPENAI_API_KEY`
- 将 `LLM_PROVIDER` 改为 `openai`

---

## 2. 飞书 Webhook URL

**步骤：**
1. 打开飞书，创建或进入任意群聊
2. 点击右上角「设置」→「机器人」→「添加机器人」
3. 选择「自定义机器人」，填写名称（如：Vibe Coding 助手）
4. 复制生成的 Webhook 地址
5. 填入 `.env` 的 `FEISHU_WEBHOOK_URL`

> 注意：Webhook 地址格式为 `https://open.feishu.cn/open-apis/bot/v2/hook/xxxxxxxx`

---

## 3. Python 环境

- Python 3.10 或以上版本
- 运行 `pip install -r requirements.txt` 安装依赖

---
